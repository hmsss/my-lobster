# 《十二祖巫都是我弟子》人物设定

## 一、主角

**姓名**：顾渊（前世名）→ 今世名待定
**身份**：穿越重生者，前世为修仙界散修，陨落于巫妖大战余波
**金手指**：
- 先知记忆（知晓巫妖大战全部结局）
- 因果洞察（可隐约感知命运线波动）
- 模拟推演（可推演小范围因果走向，但有次数限制）
**核心矛盾**：知晓命运却无力抗衡 → 找到改命之法
**性格**：沉稳、算无遗策、以情动人
**处境**：重生后修为尽失（炼气期），需从头修炼并接近祖巫

## 二、十二祖巫

### 1. 帝江（空间之巫）
- **外形**：如黄囊，六足四翼，浑敦无面目
- **性格**：爽朗直接，不拘小节，爱好结交好友
- **特点**：空间神通独步天下，速度第一
- **弱点**：不善心机，易被激将
- **收服难点**：祖巫中地位最高，最难拜师
- **弟子情**：被主角以"朋友"而非"徒弟"身份打动

### 2. 句芒（东方木之巫）
- **外形**：青鸟人身，乘两龙
- **性格**：温和仁慈，重生滋养，与帝江交好
- **特点**：木系神通，登能生死人肉白骨
- **弱点**：过于仁慈，不忍杀生
- **收服难点**：帝江好友，需先打动帝江
- **弟子情**：主角以"万物皆需平衡"说服

### 3. 蓐收（西方金之巫）
- **外形**：人面虎身，白毛巨爪，持钺
- **性格**：刚正肃杀，军纪严明
- **特点**：金系杀伐神通，战力极高
- **弱点**：过于严苛，部族多有怨言
- **收服难点**：需展示主角的战略眼光
- **弟子情**：主角助其破解一场阴谋后拜服

### 4. 共工（水之巫）
- **外形**：人面蛇身，通体赤红
- **性格**：暴烈如火，但内心孤独（与祝融不睦）
- **特点**：水系神通，可引发滔天洪水
- **弱点**：与祝融世仇，常因此失控
- **收服难点**：需调解与祝融的矛盾
- **弟子情**：主角首徒，以真心换真心

### 5. 祝融（火之巫）
- **外形**：兽头人身，身披红鳞，耳穿火蛇
- **性格**：暴躁热烈，与共工世代为仇
- **特点**：火系神通，可焚天煮海
- **弱点**：易怒，与共工冲突不断
- **收服难点**：需化解与共工的误会
- **弟子情**：被主角的"不偏不倚"打动

### 6. 强良（雷之巫）
- **外形**：嘴里衔蛇，手中握蛇，虎头人身，四蹄足
- **性格**：沉默寡言，实则重视情义
- **特点**：雷系神通，攻击类神通中杀伤力最强
- **弱点**：不善言辞，常被误解冷漠
- **收服难点**：需以诚打动
- **弟子情**：主角在雷暴中救下其族人

### 7. 弇兹（风之巫）
- **外形**：人面鸟身，耳挂青蛇，手践青蛇
- **性格**：灵动飘逸，来去如风，心思难测
- **特点**：风系神通，极速机动
- **弱点**：飘忽不定，难以捉摸
- **收服难点**：需跟上她的节奏
- **弟子情**：主角以"风的自由亦有归处"说服

### 8. 烛龙（时间之巫）
- **外形**：人面蛇身，通体赤红，睁眼为昼，闭眼为夜
- **性格**：苍老沉稳，知晓部分天机
- **特点**：时间神通，可短暂操控昼夜
- **弱点**：因窥天机，反噬缠身
- **收服难点**：需展示主角知晓更多天机
- **弟子情**：主角展示的命运知识令其震惊

### 9. 后土（大地之巫）
- **外形**：雍容华贵，身披黄衣，气度深不可测
- **性格**：宽厚仁慈，悲天悯人，后为轮回之母
- **特点**：土系神通，大地之力无出其右
- **弱点**：悲悯心太重，常舍己为人
- **收服难点**：需打动其慈悲心
- **弟子情**：主角提出"以巫族方式救赎亡魂"打动她

### 10. 奢比尸（天气之巫）
- **外形**：人头兽身，双耳似羊，披挂各种天气之力
- **性格**：懒散嗜睡，不问事务，实则大智若愚
- **特点**：能操控天气，呼风唤雨
- **弱点**：过于慵懒，不在乎胜负
- **收服难点**：需以"有趣"打动
- **弟子情**：主角的推演游戏让他感到有趣

### 11. 天吴（水风双系巫）
- **外形**：八首人面，虎身八足，水风交织
- **性格**：多面复杂，难以捉摸
- **特点**：水风双系，可引发风暴海啸
- **弱点**：性格分裂，难以统一
- **收服难点**：需接受其多面性
- **弟子情**：主角的因果洞察看到了真实的他

### 12. 龠兹（音律之巫）
- **外形**：女子形态，绝美，以龠为饰
- **性格**：高傲艺术，不谐世事
- **特点**：音律神通，可杀人无形，可疗愈心灵
- **弱点**：活在自己世界里
- **收服难点**：需展示对音律的理解
- **弟子情**：主角以一曲凡音勾起其共鸣

## 三、主要配角

### 妖族
- **帝俊**：妖庭之主，城府极深，野心勃勃，以为自己能掌控命运
- **东皇太一**：妖庭柱石，战力滔天，重情重义，与帝俊理念有分歧
- **羲和**：帝俊正妻，金乌之母，温柔而坚定
- **常羲**：帝俊侧妻，嫦娥之母，月之巫妖
- **白泽**：十大妖圣之首，智慧型，文臣之首
- **鲲鹏**：妖师，阴险狡诈，暗藏野心

### 修士（教统萌芽）
- **元始**：阐教之主，高傲贵气，看重根脚
- **通天**：截教之主，率性而为，有教无类
- **接引**：西方教教主，苦行僧形象
- **准提**：西方教副教主，圆滑机变
- **镇元子**：散仙巨头，地仙之祖，盟友潜力

## 四、势力关系图

```
                    【天道/命运】
                         |
          ┌──────────────┼──────────────┐
          │              │              │
       【巫族】        【妖族】       【人族】
     十二祖巫        妖庭帝俊        散修部落
        |              |               |
      大巫           妖圣             初民
        |              |               |
     各部落         周天星斗        炼气士
```

**主角立场**：以第三方身份介入巫妖之争，不偏不倚，目标保全十二祖巫

**核心关系张力**：
- 共工 vs 祝融：世仇
- 帝俊 vs 东皇太一：理念冲突（帝俊想称霸，东皇重情义）
- 巫族 vs 妖族：争夺天地主角
- 主角 vs 命运：知晓结局却想改写
