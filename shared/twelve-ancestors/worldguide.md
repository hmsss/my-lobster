# 《十二祖巫都是我弟子》世界观设定

## 一、时代背景

**时间线**：巫妖大战前夕（距正式大劫约数万元会）
**地点**：三十三天外至幽冥地府之间的整个洪荒宇宙
**历史定位**：继龙汉大劫之后的又一次量劫前夕，天地间巫妖两族分庭抗礼，人族初兴，修士隐于山林

### 上古纪年
- 龙汉大劫（终结）→ 巫妖并立时代（当前）→ 巫妖大战（预定结局）→ 封神大劫（后续伏笔）
- 一元 = 129600年；一运 = 4320年；一会 = 1080年
- 当前处于巫妖量劫酝酿期，大战爆发尚有万余年

## 二、修炼体系

| 境界 | 特征 | 地位 |
|------|------|------|
| 炼气 | 吸纳天地灵气 | 凡人起点 |
| 筑基 | 灵气入体凝基 | 散修入门 |
| 金丹 | 丹成金光 | 小有所成 |
| 元婴 | 婴成神现 | 一方强者 |
| 金仙 | 金性不灭 | 大能门槛 |
| 大罗 | 跳出命运 | 一族支柱 |
| 准圣 | 准教主级 | 巅峰强者 |
| 圣人 | 天道化身 | 不可撼动 |

**巫族特殊**：巫族不修元神，以肉身证道，境界参照"大巫（≈金仙至大罗）、祖巫（≈准圣至圣人下最强）"

## 三、主要势力

### 巫族
- **十二祖巫**：帝江（空间）、共工（水）、祝融（火）、蓐收（金）、句芒（木）、后土（土）、强良（雷）、弇兹（风）、烛龙（时间）、奢比尸（天气）、天吴（水风）、龠兹（音律）
- **大巫**：刑天、后羿、夸父、蚩尤等，战力约等于金仙至大罗
- **巫族部落**：散布洪荒，以祖巫为核心，形成十二股势力

### 妖族
- **妖庭**：帝俊、东皇太一建立，以周天星斗大阵为镇族之宝
- **妖圣**：白泽、鲲鹏、九婴等十大妖圣
- **散妖**：不附妖庭的散修妖族

### 人族
- 刚刚脱离蒙昧，建立部落，以火种、文字为标志
- 散修修士初现，多以"炼气士"形式隐居

### 修士势力（教统萌芽）
- 阐教：元始天尊尚在玉京山清修，尚未立教，但已有收徒之意
- 截教：通天教主仍在昆仑，收徒不论根脚
- 西方教：接引、准提在西方贫瘠之地苦修，暂无大动作
- 散修联盟：散仙岛屿的一些老牌金仙形成松散联盟

## 四、量劫设定

### 巫妖量劫
- **起因**：巫妖两族争夺天地主角之位，积怨已深
- **导火索**：巫族炼制屠巫剑（以人族魂魄为祭），妖族屠巫反击
- **结局**：两败俱伤，祖巫或死或隐，妖庭崩塌，人族借势崛起
- **天道定数**：这是不可逆的命运洪流

### 封神大劫（后续伏笔）
- 巫妖战后，天道转向人族大兴
- 封神量劫实为仙道势力重新洗牌，与巫妖之战有隐秘关联

## 五、地理格局

- **三十三天**：妖庭所在，天庭雏形
- **洪荒大陆**：巫族与人族聚居
- **四海**：龙族残余势力，形式上附妖庭
- **幽冥地府**：后土化轮回之前，血海冥河老祖盘踞
- **万寿山/五庄观**：镇元子隐居，人参果树为后天灵根
- **昆仑山**：阐截两教萌芽之地
- **西方贫瘠**：接引准提道场

## 六、关键事件时间轴

1. **主角重生** → 获得先知优势，开始布局
2. **接触第一位祖巫** → 建立信任
3. **十二祖巫陆续拜入门下** → 势力成型
4. **巫妖矛盾激化** → 主角暗中周旋
5. **大战爆发** → 主角试图逆转命运
6. **量劫终结** → 祖巫存续，新格局形成
