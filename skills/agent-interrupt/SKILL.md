# Agent Interrupt Skill - 中断 Agent 任务

快速中断正在运行的 Agent 任务，查看员工工作状态。

## 触发词

- `zd` - 中断当前正在运行的 Agent
- `zd -all` - 中断所有正在运行的 Agent
- `ai` - 查看所有员工详细工作状态
- `中断`
- `stop`
- `kill`

## 使用方式

### 中断任务
```
zd              # 中断当前任务
zd -all          # 中断所有任务
zd product-manager  # 中断产品经理
```

### 查看状态
```
ai              # 查看所有员工详细工作状态（推荐）
zd -status        # 查看状态（简洁版）
```

## 快捷命令

| 命令 | 功能 |
|------|------|
| `/ai` | 查看所有员工详细工作状态 |
| `/zd` | 中断当前 Agent |
| `/zd -all` | 中断所有 Agent |
| `/zd -status` | 查看状态（简洁） |
