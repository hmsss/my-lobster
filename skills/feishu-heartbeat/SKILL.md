# feishu-heartbeat

飞书实时进度推送。在执行多步骤任务时，每完成一个关键步骤向飞书发送进度消息。

## 使用方法

使用 `message` 工具发送进度：

| 格式 | 说明 |
|------|------|
| 🔄 [n/N] 消息 | 进行中 |
| ✅ [N/N] 消息 | 完成 |

## 关键节点

| 操作 | 消息示例 |
|------|---------|
| 开始分析 | 🔄 [1/N] 正在分析任务... |
| 检查员工 | 🔄 [2/N] 正在检查现有团队... |
| 搜索人才 | 🔄 [3/N] 正在搜索人才市场... |
| 部署员工 | 🔄 [X/N] 正在部署... |
| 完成任务 | ✅ [N/N] 任务完成！ |
