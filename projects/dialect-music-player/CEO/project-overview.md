# 项目总览：方言音乐播放器

## 项目信息
- **项目代号**：dialect-music-player
- **类型**：Web 前端应用
- **发起人**：韩猛
- **开始时间**：2026-03-19

## 项目目标
开发一个 Web 版本的音乐播放器，支持：
1. **音乐播放**：播放各类音乐
2. **方言播放**：播放中国各省市县的方言音频

## 核心功能
### 模块一：音乐播放
- 音乐列表展示
- 播放/暂停/上一首/下一首
- 进度条控制
- 音量调节

### 模块二：方言播放
- 省市县三级联动选择器
- 地区方言音频列表
- 播放控制（同音乐模块）
- 方言音频数据源（需产品调研）

## 技术栈（待确认）
- 前端框架：React / Vue
- UI 组件库：Ant Design / Element Plus
- 音频播放：Howler.js / HTML5 Audio API
- 地区数据：省市区数据 JSON
