## 1. 项目概述

### 1.1 项目目标
开发一个功能完整的 Web 数独游戏，支持多种难度等级、实时校验、笔记模式、提示功能和进度保存。

### 1.2 核心功能
- 数独生成（支持 4 难度等级：简单/中等/困难/专家）
- 实时数字校验
- 笔记模式（候选数）
- 撤销/重做
- 提示功能
- 计时器 + 暂停
- 进度保存（LocalStorage）
- 完成动画
