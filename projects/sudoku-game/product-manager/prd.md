## 1. 背景与问题陈述

### 为什么做这个产品？
数独是全球最受欢迎的逻辑益智游戏之一，老少皆宜。开发一款精美的 Web 数独游戏，让用户随时随地享受解谜乐趣，同时提供辅助功能帮助新手入门。

### 用户痛点
1. **新手玩家**：不知道怎么玩，需要提示和教学
2. **进阶玩家**：需要挑战更高难度，记录成绩
3. **通勤用户**：想在手机上玩，但很多数独游戏体验差

### 目标用户
- 18-55 岁逻辑游戏爱好者
- 想锻炼脑力的用户
- 通勤/休闲时玩游戏的用户

## 3. Non-Goals（不做的事）

| 不做的事 | 原因 | 重新考虑条件 |
|----------|------|--------------|
| 多人对战 | 数独是单人游戏，对战意义不大 | 用户强烈需求 |
| 付费会员 | 先验证产品价值 | 日活 > 1000 |
| 变体数独（杀手数独等） | 增加复杂度 | 经典模式稳定后 |
| 社交分享 | 非核心功能 | 用户反馈需要 |
| 成就系统 | 非核心功能 | V2 迭代 |
| 暗黑模式 | 简化 V1 | 用户反馈 |

## 4. 游戏玩法设计

### 4.1 数独规则

```
┌─────────────────────────────────────────────────────┐
│                   数独游戏规则                       │
├─────────────────────────────────────────────────────┤
│                                                     │
│ 标准数独：9×9 方格，填入 1-9 数字                   │
│                                                     │
│ 规则：                                              │
│ 1. 每行必须包含 1-9，不重复                        │
│ 2. 每列必须包含 1-9，不重复                        │
│ 3. 每个 3×3 宫格必须包含 1-9，不重复               │
│                                                     │
│ 游戏开始：                                          │
│ • 部分格子已填入数字（提示数）                      │
│ • 玩家根据逻辑推理填满所有空格                      │
│ • 每个数独只有唯一解                                │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 4.2 难度分级

| 难度 | 提示数 | 平均解题时间 | 适合人群 |
|------|--------|--------------|----------|
| 简单 | 36-45 个 | 5-10 分钟 | 新手入门 |
| 中等 | 30-35 个 | 10-20 分钟 | 有经验玩家 |
| 困难 | 25-29 个 | 20-40 分钟 | 高级玩家 |
| 专家 | 17-24 个 | 40+ 分钟 | 大师级玩家 |

**难度设计原则**：
- 提示数越少，难度越高
- 高难度需要更复杂的推理技巧
- 保证每题都有唯一解

### 4.3 辅助功能

#### 功能 1：实时校验
```
开启后，错误填入的数字会标红提示
• 同行/同列/同宫格有重复数字 → 标红
• 可选择开启/关闭
• 不影响最终评分（仅提示）
```

#### 功能 2：笔记模式
```
在格子中记录候选数字
• 点击"笔记"按钮进入笔记模式
• 可在空格中标记多个候选数字（小字显示）
• 帮助玩家推理和排除
• 笔记不影响游戏完成判定
```

#### 功能 3：提示功能
```
当玩家卡住时，提供提示
• "显示一个正确数字"：随机填入一个正确答案
• 使用提示会扣除分数/时间惩罚
• 每局最多使用 3 次提示
```

#### 功能 4：撤销/重做
```
支持撤销和重做操作
• 撤销：恢复上一步操作
• 重做：恢复被撤销的操作
• 最多保存 50 步历史
```

#### 功能 5：高亮相同数字
```
点击某数字时，高亮所有相同数字
• 帮助玩家快速定位和排除
• 视觉效果：同数字格子背景变色
```

## 5. 用户故事

### 故事 1：开始新游戏
**作为** 一名玩家
**我希望** 能快速开始一局新游戏
**以便** 立即开始游玩

**验收标准**：
- [ ] 主页显示"新游戏"按钮
- [ ] 可选择难度（简单/中等/困难/专家）
- [ ] 选择后立即生成数独题目
- [ ] 显示初始提示数字（不可修改）

### 故事 2：填写数字
**作为** 一名玩家
**我希望** 能方便地填入数字
**以便** 完成数独

**验收标准**：
- [ ] 点击空格选中该格
- [ ] 点击数字键盘填入数字
- [ ] 支持键盘 1-9 快捷输入
- [ ] 支持删除/Backspace 清空
- [ ] 提示数字（初始数字）不可修改

### 故事 3：使用笔记
**作为** 一名玩家
**我希望** 能在格子中记录候选数字
**以便** 辅助推理

**验收标准**：
- [ ] 有"笔记"模式开关
- [ ] 笔记模式下点击格子可标记候选数
- [ ] 候选数以小字显示在格子内
- [ ] 可取消笔记标记
- [ ] 填入数字时自动清除该格笔记

### 故事 4：查看提示
**作为** 一名新手玩家
**我希望** 卡住时能获得提示
**以便** 继续游戏

**验收标准**：
- [ ] 有"提示"按钮
- [ ] 点击后随机填入一个正确数字
- [ ] 显示剩余提示次数（最多 3 次）
- [ ] 使用提示有时间惩罚（+30秒）

### 故事 5：撤销操作
**作为** 一名玩家
**我希望** 能撤销错误的输入
**以便** 纠正错误

**验收标准**：
- [ ] 有"撤销"按钮
- [ ] 撤销后恢复上一步状态
- [ ] 支持"重做"恢复撤销
- [ ] 撤销也适用于笔记操作

### 故事 6：查看进度
**作为** 一名玩家
**我希望** 知道我完成了多少
**以便** 了解游戏进度

**验收标准**：
- [ ] 显示已填格子数 / 总格子数
- [ ] 显示进度条
- [ ] 显示当前用时
- [ ] 显示已用提示次数

### 故事 7：完成游戏
**作为** 一名玩家
**我希望** 完成后能看到成绩
**以便** 获得成就感

**验收标准**：
- [ ] 填满所有格子后自动校验
- [ ] 正确完成显示"恭喜完成"
- [ ] 显示完成时间
- [ ] 显示使用提示次数
- [ ] 可选择"再来一局"或"返回主页"

### 故事 8：每日挑战
**作为** 一名玩家
**我希望** 每天有一个固定题目
**以便** 和朋友比较成绩

**验收标准**：
- [ ] 每天更新一道挑战题
- [ ] 同一天所有玩家看到相同题目
- [ ] 完成后显示排名（本地）
- [ ] 显示完成率

### 故事 9：暂停游戏
**作为** 一名玩家
**我希望** 能暂停计时器
**以便** 暂时离开

**验收标准**：
- [ ] 有"暂停"按钮
- [ ] 暂停后计时停止
- [ ] 暂停时隐藏题目（防作弊）
- [ ] 点击"继续"恢复游戏
