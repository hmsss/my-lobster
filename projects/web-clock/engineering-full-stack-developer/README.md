# Web 时钟

## 运行方式

直接在浏览器中打开 `index.html` 即可。

## 已实现功能

### P0 核心功能
- ✅ 模拟时钟（圆形表盘 + 指针动画）
- ✅ 数字时钟（24/12 小时制切换）
- ✅ 当前日期显示
- ✅ 世界时钟（多城市）
- ✅ 闹钟（添加/开关/删除）
- ✅ 秒表（精确到 0.01 秒 + 计次）
- ✅ 倒计时（预设 + 自定义）
- ✅ 番茄钟（25分钟工作/5分钟休息循环）

### P1 重要功能
- ✅ 深色/浅色主题切换
- ✅ 显示秒数开关
- ✅ 闹钟响铃提示

## 操作说明

| 功能 | 操作 |
|------|------|
| 切换标签 | 点击顶部标签页 |
| 时钟格式 | 设置中切换 24/12 小时 |
| 添加闹钟 | 闹钟页 → 点击添加 |
| 秒表计次 | 运行中点击"计次" |
| 倒计时 | 选择预设或输入分钟 |

## 技术实现

- **框架**: Vue 3 (CDN)
- **样式**: 原生 CSS
- **存储**: LocalStorage
- **动画**: CSS + requestAnimationFrame
