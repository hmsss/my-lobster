# Web 时钟 - 项目总览

## 项目目标
开发一款精美的 Web 时钟应用，支持世界时钟、闹钟、秒表、计时器等功能。

## 项目范围
- **前端**：响应式时钟界面
- **功能**：
  - 实时时钟显示（模拟/数字）
  - 世界时钟（多城市）
  - 闹钟提醒
  - 秒表
  - 倒计时计时器
  - 番茄钟模式
  - 主题切换

## 技术栈
- 前端：Vue 3 + TypeScript + CSS Animation
- 存储：LocalStorage（闹钟设置）

## 预期产出
1. 可用的时钟应用
2. 完整的测试报告
