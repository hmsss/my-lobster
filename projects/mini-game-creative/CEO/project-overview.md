# 弹跳吃吃 Bounce & Eat - 项目总览

## 项目目标
开发一款简单易上手但挑战无极限的弹跳小游戏

## 游戏概述
控制一个不断弹跳的小球，按住屏幕吸附控制，松开继续弹跳，吃掉食物得分，躲避障碍

## 核心功能
- 小球持续弹跳物理
- 点击吸附控制机制
- 食物系统（普通+特殊）
- 障碍物系统
- 三种游戏模式
- 成长升级系统

## 技术栈
- 纯前端：HTML5 Canvas + JavaScript
- 无需后端

## 预期产出
1. 可玩的 HTML5 小游戏
2. 完整 PRD 文档
