# 弹跳吃吃 Bounce & Eat

## 运行方式

直接在浏览器中打开 `index.html` 即可。

## 游戏特色

### 🎮 吸附控制
- 按住屏幕/鼠标：球向你按住的位置移动
- 松开：球按当前方向继续弹跳
- 有牵引线视觉效果

### 三种游戏模式

| 模式 | 说明 |
|------|------|
| ♾️ 无尽模式 | 3条命，障碍递增，挑战高分 |
| ⏱️ 限时90秒 | 无命限制，90秒内拿最高分 |
| 🧘 禅模式 | 无危险无时间，纯粹享受 |

### 🍎 食物系统

| 食物 | 效果 |
|------|------|
| 💡 黄点 | +10分 |
| 💡 大黄点 | +25分 |
| 🌈 彩虹 | +50分 + 清屏 |
| 🛡️ 护盾 | 5秒无敌 |
| 🐢 减速 | 障碍减速5秒 |
| ❤️ 生命 | +1条命 |

### ⬆️ 等级系统

| 等级 | 食物数 | 球大小 | 特效 |
|------|--------|--------|------|
| Lv.1 | 0 | 15px | 白色 |
| Lv.2 | 5 | 18px | 绿色光晕 |
| Lv.3 | 12 | 21px | 蓝色光晕 |
| Lv.4 | 20 | 24px | 紫色光晕 |
| Lv.5 | 30 | 27px | 金色+拖尾 |

### 🔺 障碍物

| 障碍 | 效果 |
|------|------|
| 红色尖刺 | 扣1条命 |
| 蓝色方块 | 可撞碎 +20分 |

## 操作

- **PC**：按住鼠标 / 松开
- **手机**：按住屏幕 / 松开
- 按空格可快速开始（无尽模式）
