## 1. 背景与游戏定位

### 为什么做这个游戏？
弹跳类游戏天然具有「易上手」的特性，但《弹跳吃吃》通过独特的「吸附控制」机制，让简单中带技巧，同时霓虹视觉风格提供强烈的感官刺激，适合休闲解压。

### 核心特色
| 特色 | 说明 |
|------|------|
| 🎮 独特操作 | 按住吸附控制，打破传统方向键/触屏滑动 |
| 📈 无限挑战 | 每次游玩体验不同，技巧无上限 |
| ⬆️ 成长系统 | 吃够食物升级变大，体验「变强」快感 |
| 🌈 霓虹视觉 | 暗色背景 + 霓虹发光效果，炫酷吸睛 |

### 目标用户
- 休闲玩家：想找一款随时能玩、随时停下的游戏
- 挑战型玩家：追求高分、高技巧展示
- 视觉党：喜欢炫酷视觉效果

## 4. 成长系统

### 4.1 等级设计

| 等级 | 所需食物 | 球半径 | 速度加成 | 特效 |
|------|----------|--------|----------|------|
| Lv.1 | 0 | 15px | 0% | 白色核心 |
| Lv.2 | 5 | 18px | +5% | 绿色光晕 |
| Lv.3 | 12 | 21px | +10% | 蓝色光晕 |
| Lv.4 | 20 | 24px | +15% | 紫色光晕 |
| Lv.5 | 30 | 27px | +20% | 金色光晕 + 拖尾 |

### 4.2 升级效果

- **越大越好**：更容易吃到食物，撞障碍更稳
- **越快越好**：弹跳更快，操作更灵活
- **视觉反馈**：升级时屏幕闪光 + 音效

```
• 食物被吃掉后，在随机位置生成新食物
```

## 7. 数值设计

### 7.1 得分体系

| 动作 | 得分 | 说明 |
|------|------|------|
| 吃普通光点 | +10 | 基础食物 |
| 吃大光点 | +25 | 较大食物 |
| 吃彩虹食物 | +50 | 稀有食物 |
| 撞碎蓝色方块 | +20 | 可摧毁障碍 |
| 使用护盾存活 | +5/秒 | 护盾期间每秒 |
| 升级 | +30 | 每升一级 |

### 7.2 惩罚体系

| 动作 | 惩罚 | 说明 |
|------|------|------|
| 撞尖刺 | -1 条命 | 直接生命损失 |
| 撞蓝色方块 | 无惩罚 | 可撞碎 |

### 7.3 难度平衡

```
目标：让休闲玩家能玩 60 秒以上，高手能挑战 3 分钟+

• 休闲玩家：靠走位躲避障碍，存活 60-90 秒
• 中级玩家：利用升级，存活 90-180 秒
• 高手：精准吸附控制 + 撞方块得分，存活 180 秒+
```
