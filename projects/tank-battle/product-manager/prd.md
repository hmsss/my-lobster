## 1. 背景与问题陈述

### 为什么做这个产品？
坦克大战（Battle City）是经典的 FC 游戏，承载了 80/90 后的童年回忆。开发一款 Web 版坦克大战，让玩家可以在浏览器中重温经典，同时支持双人对战，增加社交乐趣。

### 用户痛点
1. **怀旧玩家**：想在现代设备上重温经典游戏
2. **社交需求**：想和朋友一起玩对战游戏
3. **便捷性**：不想下载安装，打开浏览器就能玩

### 目标用户
- 25-40 岁怀旧玩家
- 休闲游戏爱好者
- 想和朋友一起玩的人

## 2. 目标与成功指标

| 目标 | 指标 | 目标值 | 度量窗口 |
|------|------|--------|----------|
| 用户留存 | 次日留存率 | > 20% | 上线后 7 天 |
| 游戏时长 | 平均游戏时长 | > 10 分钟 | 上线后 7 天 |
| 双人参与 | 双人对战占比 | > 30% | 上线后 14 天 |

## 3. Non-Goals（不做的事）

| 不做的事 | 原因 | 重新考虑条件 |
|----------|------|--------------|
| 移动端原生 App | V1 聚焦 Web 端 | PC 版成功后 |
| 4 人以上对战 | 技术复杂度高 | 双人模式稳定后 |
| 付费道具 | 先验证产品价值 | 日活 > 1000 |
| 成就系统 | 非核心功能 | 用户反馈需要 |
| 回放系统 | 存储成本高 | 有强烈需求 |
| 语音聊天 | 依赖第三方服务 | 可用第三方 |

## 4. 游戏玩法设计

### 4.1 游戏模式

#### 模式一：单人闯关（PVE）
```
┌─────────────────────────────────────────────────────┐
│              单人闯关模式                            │
├─────────────────────────────────────────────────────┤
│ 玩家控制一辆坦克，消灭所有敌人，保护基地            │
│                                                     │
│ • 关卡制：共 35 关（经典关卡）                      │
│ • 生命：初始 3 条命                                │
│ • 敌人：每关出现 20 辆敌方坦克                      │
│ • 胜利条件：消灭所有敌人且基地未被摧毁              │
│ • 失败条件：生命耗尽 或 基地被摧毁                  │
│                                                     │
│ 敌方坦克类型：                                      │
│ • 普通坦克（灰色）：1 发子弹击毁                    │
│ • 快速坦克（白色）：移动快，1 发击毁                │
│ • 装甲坦克（绿色）：需 3 发子弹击毁                 │
│ • 超级坦克（红色）：需 3 发击毁，有护盾             │
└─────────────────────────────────────────────────────┘
```

#### 模式二：双人闯关（PVE Co-op）
```
┌─────────────────────────────────────────────────────┐
│              双人闯关模式                            │
├─────────────────────────────────────────────────────┤
│ 两名玩家各控制一辆坦克，合作消灭敌人                │
│                                                     │
│ • 玩家 1：黄色坦克，WASD + 空格射击                 │
│ • 玩家 2：绿色坦克，方向键 + Enter 射击             │
│ • 共享生命：3 条命（任意玩家死亡扣 1 条）           │
│ • 友军伤害：关闭（子弹穿过队友）                    │
│ • 胜利/失败条件：同单人模式                         │
└─────────────────────────────────────────────────────┘
```

#### 模式三：双人对战（PVP）
```
┌─────────────────────────────────────────────────────┐
│              双人对战模式                            │
├─────────────────────────────────────────────────────┤
│ 两名玩家互相对战，先击败对方 5 次获胜              │
│                                                     │
│ • 地图：专门的竞技场地图（无基地）                  │
│ • 胜利条件：先击败对方 5 次                         │
│ • 道具：地图随机刷新道具                            │
│ • 模式：                                            │
│   - 本地对战：同一台电脑                            │
│   - 在线对战：WebSocket 实时同步                    │
└─────────────────────────────────────────────────────┘
```

### 4.2 游戏规则

#### 移动规则
| 元素 | 移动方式 |
|------|----------|
| 玩家坦克 | 4 方向移动（上下左右），不能斜向 |
| 敌方坦克 | AI 控制，4 方向移动 |
| 子弹 | 直线飞行，碰到障碍物或边界消失 |

#### 碰撞规则
| 碰撞场景 | 结果 |
|----------|------|
| 子弹击中玩家坦克 | 扣 1 条命，坦克重生 |
| 子弹击中敌方坦克 | 敌方坦克被摧毁 |
| 子弹击中砖墙 | 砖墙被摧毁 |
| 子弹击中钢墙 | 子弹消失（钢墙无损） |
| 子弹击中基地 | 游戏失败 |
| 子弹击中子弹 | 两颗子弹同时消失 |
| 坦克撞坦克 | 无法通过，互相阻挡 |

#### 地图元素
| 元素 | 图例 | 说明 |
|------|------|------|
| 空地 | □ | 可自由通行 |
| 砖墙 | ░ | 可被子弹摧毁 |
| 钢墙 | █ | 不可摧毁 |
| 草地 | ▒ | 隐藏坦克（可穿过） |
| 水域 | ≈ | 无法通过 |
| 基地 | ★ | 被击中则游戏失败 |
| 冰面 | ◇ | 滑行效果 |

### 4.3 道具系统

| 道具 | 效果 | 持续时间 |
|------|------|----------|
| ⭐ 星星 | 坦克升级（速度+射速） | 永久（本关） |
| 🛡️ 护盾 | 免疫伤害 | 10 秒 |
| 💣 炸弹 | 消灭屏幕所有敌人 | 立即 |
| ⏰ 时钟 | 敌人停止移动 | 10 秒 |
| 🔨 铲子 | 基地周围变成钢墙 | 永久（本关） |
| ❤️ 生命 | +1 条命 | 永久 |

### 4.4 坦克升级系统

| 等级 | 速度 | 射速 | 子弹数 |
|------|------|------|--------|
| Lv.1 | 正常 | 1 发/秒 | 1 发 |
| Lv.2 | +20% | 2 发/秒 | 1 发 |
| Lv.3 | +30% | 2 发/秒 | 2 发 |

收集 1 个星星升 1 级，最高 Lv.3，死亡后重置为 Lv.1。

## 5. 用户故事

### 故事 1：开始单人游戏
**作为** 一名玩家
**我希望** 能快速开始一局单人游戏
**以便** 独自享受游戏乐趣

**验收标准**：
- [ ] 主页显示"单人游戏"按钮
- [ ] 点击后选择关卡（1-35 关）
- [ ] 进入游戏，控制黄色坦克
- [ ] 显示当前关卡、分数、生命数

### 故事 2：双人本地对战
**作为** 一名玩家
**我希望** 和朋友在同一台电脑上对战
**以便** 一起玩游戏

**验收标准**：
- [ ] 主页显示"双人对战"按钮
- [ ] 玩家 1 使用 WASD + 空格
- [ ] 玩家 2 使用方向键 + Enter
- [ ] 显示双方得分
- [ ] 先击败对方 5 次获胜

### 故事 3：在线对战
**作为** 一名玩家
**我希望** 和远方的朋友在线对战
**以便** 远程一起玩

**验收标准**：
- [ ] 创建房间，获得房间码
- [ ] 朋友输入房间码加入
- [ ] 双方准备后开始游戏
- [ ] 实时同步，延迟 < 100ms
- [ ] 断线重连功能

### 故事 4：使用道具
**作为** 一名玩家
**我希望** 捡到道具获得增益效果
**以便** 更容易获胜

**验收标准**：
- [ ] 道具随机出现在地图上
- [ ] 坦克碰到道具自动拾取
- [ ] 显示道具效果提示
- [ ] 护盾有视觉反馈（发光）

### 故事 5：暂停游戏
**作为** 一名玩家
**我希望** 能暂停游戏
**以便** 临时离开

**验收标准**：
- [ ] 按 ESC 或 P 键暂停
- [ ] 显示暂停菜单（继续、重新开始、退出）
- [ ] 游戏时间停止

### 故事 6：查看排行榜
**作为** 一名玩家
**我希望** 查看高分排行榜
**以便** 和其他玩家比较

**验收标准**：
- [ ] 主页显示排行榜入口
- [ ] 显示前 100 名分数
- [ ] 显示玩家排名（如有）
- [ ] 支持按关卡/总分查看

### 故事 7：使用地图编辑器
**作为** 一名玩家
**我希望** 自定义地图
**以便** 创造独特玩法

**验收标准**：
- [ ] 可选择地图元素（砖墙、钢墙等）
- [ ] 可保存地图
- [ ] 可加载自定义地图游戏
- [ ] 可分享地图码

## 9. 音效设计

| 音效 | 触发时机 |
|------|----------|
| 射击 | 发射子弹时 |
| 爆炸 | 坦克/墙壁被摧毁时 |
| 道具 | 拾取道具时 |
| 移动 | 坦克移动时（可选，有音量控制） |
