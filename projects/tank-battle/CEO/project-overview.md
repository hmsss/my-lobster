# Web 坦克大战 - 项目总览

## 项目目标
开发一款基于 Web 的坦克大战游戏，支持单人/双人模式，具备经典玩法和现代视觉效果。

## 项目范围
- **前端**：基于 Canvas/WebGL 的游戏引擎
- **后端**：实时对战服务器（WebSocket）
- **功能**：
  - 单人模式（AI 对战）
  - 双人模式（本地/在线）
  - 地图编辑器
  - 排行榜系统
  - 音效和特效

## 技术栈
- 前端：TypeScript + Canvas/WebGL + Vite
- 后端：Node.js + Socket.io + Redis
- 部署：Docker + Nginx
