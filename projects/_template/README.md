# Project Template

Copy this directory to `projects/{project_slug}/` when starting a new project.

## Included

- `docs/` for project documentation
- `memory/MEMORY.md` for long-term project memory
- `memory/SESSION-STATE.md` for current project state
- `reports/` for delivery and status artifacts
- `automation/` for scripts, fixtures, and reusable project automation

## Rules

- Replace `_template` with the real `project_slug`
- Keep all project-specific facts inside the project directory
- Update project memory continuously during work
- Do not store human-private memory here unless it is required for the project itself
